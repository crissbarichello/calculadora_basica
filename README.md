# CALCULADORA BASICA

## Plano de Teste 

## 1. Objetivo dos Testes

- Validar funcionalidades de operações básicas de uma calculadora que devem conter;
    * Somar
    * Dividir
    * Subtrair
    * Multiplicar

- Verificar requisitos mínimos necessários para atender ao projeto;
    * Número de 0 a 9;
    * Permitir somente números;
    * Conter os operadores listados;
    * Mostrar resultados das operações;
    * Limpar resultados após final de cada operação;
    * permitir somente imputs em campos do tipo

## 2. Metodologia de Teste

- Metologias aplicadas serão Teste Exploratório e Teste de caixa Branco, por termo acesso ao projeto.

## 3. Escopo e Testes

- O teste deve abranger as funcionalidades básicas essencias de uma calculadora básica, validando as operações individuais, também validar tipos de dados permitidos e tratativas de erros.

## 4. Tipos de Testes Aplicados

- Teste reslizados serão do tipo Unidade, testando cada funcionalidade de forma individual, testando cada funcão e avaliando seu resultado.

## 5. Forma de Execução

- Os Teste serão executados neste primeiro momento de forma manual, validando cada tipo de operação e inserção de dados.

## 6. Ambiente de Teste

- O Ambiente onde será executado contém dois navegadores mais usados como Google Chorme e Microsoft Edge, em um hardware padrão com ambiente com sistema operacional Windows 11 x64.

## 7. Papéis e Responsabilidades

- Testes executados a nível de Desenvolvimento pelo Dev.

## 8. Riscos e Contingências

- Possibilidade de cáculos erradados e ações indevidas por bugs. Foco inicial em validações de resultados das operações.

## 9. Ferramentas Teste

-Será utilizado navegadores e Visual Studio Code como ferramenta de teste.
